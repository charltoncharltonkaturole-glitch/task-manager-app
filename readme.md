# 📋 Task Manager App

A full-featured task management application with user authentication, admin dashboard, dark mode, and contact messaging.

## ✨ Features

- **User Authentication** - Register, login, and session management
- **Task Management** - Create, complete, delete, and filter tasks
- **Admin Dashboard** - View visitor analytics, manage messages, user management
- **Contact System** - Users can send messages to admin
- **Dark Mode** - Toggle between light and dark themes
- **Visitor Tracking** - Track page visits and unique visitors
- **Responsive Design** - Works on desktop, tablet, and mobile

## 🛠️ Tech Stack

- **Backend:** Node.js, Express.js
- **Database:** MySQL
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Authentication:** Session-based with bcrypt
- **Styling:** Custom CSS with dark mode support
